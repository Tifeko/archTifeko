# Arch install script


**Run it using:**

``` bash
bash <(curl -L raw.githubusercontent.com/Tifeko/archTifeko/main/ArchTifeko)
```

## How to use
1. Download the latest iso from the [the arch website](https://archlinux.org/download/).
2. Then run is using:
``` bash
bash <(curl -L raw.githubusercontent.com/Tifeko/archTifeko/main/ArchTifeko)
```
